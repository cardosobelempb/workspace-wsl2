## Verificar disponibilidade

- windows 11
- windows 10 versão 1906, build 18362 ou superior.
- obs: Para verificar a versão e build do seu windows execut o comando.

```basch
winver
```

## Instalação wsl2

```basch
wsl --install
wsl --help
wsl.exe --help
```

## Revomendo wsl2

```basch
wsl --unregister <DistributionName>
wsl --unregister Debian
```
